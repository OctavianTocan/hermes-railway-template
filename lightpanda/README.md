# Lightpanda Browser Service

Headless browser on Railway for AI-driven browser automation. Uses
[Lightpanda](https://github.com/lightpanda-io/browser) — a Zig-based
headless browser that is 11x faster and uses 9x less memory than Chrome.

Exposes Chrome DevTools Protocol (CDP) on port 9222.

## Connecting

### From Hermes (Railway internal network)

```python
from playwright.async_api import async_playwright

async with async_playwright() as p:
    browser = await p.chromium.connect_over_cdp(
        "ws://lightpanda.railway.internal:9222"
    )
    page = await browser.new_page()
    await page.goto("https://example.com")
    content = await page.content()
```

### From OpenClaw (external)

```python
browser = await p.chromium.connect_over_cdp(
    "wss://<your-railway-domain>"
)
```

Replace `<your-railway-domain>` with the domain generated by Railway.

## Limitations

- **No screenshots** — Lightpanda has no rendering engine
- **Beta** — some complex SPAs may hit missing Web APIs
- **CDP only** — MCP tools are only available via Lightpanda's cloud service
